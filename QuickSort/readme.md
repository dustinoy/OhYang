# QuickSort 作業

* 問題描述
現在有一個隨機數字的list且數字順序沒有從小排到大，我們需運用QuickSort來快速整理並使其從小到大從新排列並回傳新的list。

* 程式設計架構
設定3個空的list分為left,mid,right，然後從list抓第二個數字為基準值去比較，比基準值小的數字放到left，相等的放在mid,大的放在right
